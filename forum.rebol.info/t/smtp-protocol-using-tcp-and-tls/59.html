Smtp protocol using TCP and TLS

Usage Show & Tell
gchiu July 3, 2020, 5:16pm #1

The SMTP protocol is invoked with

import <SMTP>

If you want to suppress seeing the source at the REPL you can, as with most things, just add a null evaluation with ()

import <SMTP> ()

Currently, the protocol only supports TCP, or TLS ( under version 1.2). It doesn't support STARTTLS which you might see on ports 587, and 2525. In other words, you can't yet start with TCP and then negotiate a handshake to TLS though I presume it could be done by someone knowledgeable about the prot-tls code. @Cyphre are you out there somewhere??
